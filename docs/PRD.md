# PRD: 인사 페이지 개발 프로젝트 📋

**작성자**: 멍뭉 (전략기획 담당)  
**작성일**: 2026.03.13  
**협업**: 치치(마케팅) - 동시 작업 중

---

## 1. 프로젝트 개요 & 목적 🎯

### 목적
- 개인/팀 브랜딩을 위한 전문적인 인사 페이지 개발
- 방문자에게 명확한 정체성과 역량 전달
- 연락/상담 유도를 통한 비즈니스 연결점 제공

### 타겟 유저
- 잠재 클라이언트/고객
- 협업 파트너 후보
- 채용 담당자 (경력직 대상)

---

## 2. 핵심 기능 목록 (우선순위별) ⭐

### Phase 1 (필수)
1. **헤더 네비게이션** - 페이지 섹션 이동
2. **자기소개 섹션** - 핵심 메시지 전달
3. **경력/서비스 섹션** - 전문성 어필
4. **연락처 섹션** - 컨택 포인트 제공
5. **반응형 디자인** - 모든 디바이스 대응

### Phase 2 (선택)
1. 다크모드 지원
2. 언어 전환 (한/영)
3. 포트폴리오 갤러리
4. 방문자 통계 (간단한 애널리틱스)

## 3. 화면별 기능 명세 📱💻

### 3.1 헤더 섹션
```
구성요소:
- 로고/이름 (좌측)
- 네비게이션 메뉴 (우측)
  * Home, About, Services, Contact
- 모바일: 햄버거 메뉴

기능:
- 스크롤 시 헤더 고정
- 현재 섹션 하이라이트
- 스무스 스크롤 이동
```

### 3.2 자기소개 섹션 (Hero)
```
구성요소:
- 대표 이미지/프로필 사진
- 메인 타이틀 + 서브 타이틀
- 짧은 소개글 (2-3줄)
- CTA 버튼 ("연락하기", "이력서 다운로드")

기능:
- 타이핑 애니메이션 (선택)
- 버튼 호버 효과
```

### 3.3 경력/서비스 섹션
```
구성요소:
- 섹션 제목
- 서비스/역량 카드들 (3-6개)
  * 아이콘
  * 제목
  * 설명 (2-3줄)
- 경력 타임라인 (선택)

기능:
- 스크롤 시 카드 페이드인 애니메이션
- 카드 호버 효과
```

### 3.4 연락처 섹션
```
구성요소:
- 연락 방법 (이메일, 전화, SNS)
- 간단한 연락 폼 (이름, 이메일, 메시지)
- 소셜 링크 아이콘들

기능:
- 폼 유효성 검사
- 이메일 발송 (백엔드 연동)
- 성공/실패 메시지 표시
```

## 5. API 엔드포인트 설계 🔌

```
GET /api/profile
- 프로필 정보 조회

GET /api/services
- 서비스/경력 정보 조회

POST /api/contact
- 연락 폼 제출
Body: { name, email, message }

GET /api/contact/messages (관리자용)
- 받은 메시지 목록 조회
```

## 6. 기술스택 추천 💻

### 프론트엔드
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **애니메이션**: Framer Motion
- **Icons**: Lucide React 또는 Hero Icons

### 백엔드
- **Runtime**: Node.js
- **Database**: SQLite (간단) 또는 PostgreSQL
- **이메일**: Nodemailer 또는 Resend
- **배포**: Vercel (프론트+백엔드 통합)

## 10. 유키(백엔드)에게 전달사항 🐾

**안녕 유키! 다음 작업 부탁해:**

1. **연락 폼 API 개발**
   - `POST /api/contact` 엔드포인트
   - 이메일 발송 기능 (Nodemailer 사용)
   - 간단한 스팸 방지 (rate limiting)

2. **데이터 관리**
   - 받은 메시지를 DB에 저장
   - 관리자 페이지에서 메시지 확인 가능하게

3. **환경 설정**
   - 이메일 SMTP 설정
   - 환경변수 관리 (.env)

**우선순위**: 연락 폼 > 데이터 저장 > 관리자 기능 순으로!
